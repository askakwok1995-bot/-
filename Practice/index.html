阶段 1：编程与 Web 基础认知

目标：认识编程学习方式，建立 HTML 与浏览器的基本概念。

建议时长：1-2 周

学习任务

  • 安装并熟悉编辑器（如 VS Code）与浏览器开发者工具。
  • 理解文件和文件夹、扩展名、路径这些最基础概念。
  • 学习 HTML 核心标签： html、head、body、h1-h6、p、a。
  • 学会通过双击文件或本地服务器打开网页并观察变化。

产出物

一个包含标题、段落、链接的“自我介绍”静态网页。

阶段 2：页面结构与信息表达

目标：掌握常见内容结构标签，能表达列表、数据与输入信息。

建议时长：2-3 周

学习任务

  • 学习列表标签：ul、ol、li，用来组织步骤和清单。
  • 学习表格标签：table、tr、th、td，用来展示结构化数据。
  • 学习表单标签：form、label、input、button，理解用户输入。
  • 学习语义化标签：header、nav、main、section、footer。

产出物

一个“学习计划页”，至少包含一个列表、一个表格和一个表单结构。

阶段 3：样式与交互入门预告

目标：了解后续会学习什么，建立 CSS 与 JavaScript 的整体认识。

建议时长：1 周（预习）

学习任务

  • 认识 CSS 作用：控制字体、颜色、布局、间距。
  • 认识 JavaScript 作用：实现按钮点击、表单校验、动态内容。
  • 区分结构（HTML）、样式（CSS）、行为（JavaScript）三者分工。
  • 整理你在前两阶段遇到的问题，形成自己的学习问答清单。

产出物

一份“下阶段学习准备清单”，明确将要学习的 CSS/JS 主题。

阶段 4：小项目实战路线

目标：通过小项目巩固基础，建立“做中学”的节奏。

建议时长：2-4 周

学习任务

  • 项目 1：个人主页（自我介绍、兴趣、联系方式）。
  • 项目 2：待办清单页面（先做静态结构，后续再加交互）。
  • 项目 3：学习作品集（把前面页面统一整理归档）。
  • 每个项目结束后写 3 条复盘：学到了什么、卡点是什么、下次如何改进。

产出物

一个包含多个静态页面的小型作品集目录。

阶段 5：CSS 进阶阶段

目标：掌握选择器优先级、盒模型、布局与响应式，提升页面工程能力。

建议时长：2-3 周

学习任务

  • 理解选择器优先级（标签、类、ID）与样式覆盖规则。
  • 掌握盒模型：内容区、内边距、边框、外边距。
  • 练习布局：display、flex、grid 基础。
  • 学习定位：position 与 z-index 常见用法。
  • 使用媒体查询做移动端适配。

1. 选择器优先级

观察同一元素被标签、类、ID 同时命中时的样式覆盖关系。

只有标签选择器命中（p）。

标签 + 类选择器命中（p + .note）。

标签 + 类 + ID 同时命中（优先级最高）。

2. 盒模型

每个元素都像一个盒子。下方卡片展示内容区、内边距和外边距效果。

内容区（content）

文字所在区域，由宽高控制。

内边距（padding）

内容与边框之间的留白。

外边距（margin）

盒子和盒子之间的距离。

模态框 + 键盘无障碍示例

这个示例演示 4 个关键点：打开后聚焦、Tab 焦点循环、Esc 关闭、关闭后焦点回到触发按钮。

  • 按 Tab：焦点会在弹窗内循环，不会跳到弹窗外。
  • 按 Shift + Tab：焦点会反向循环。
  • 按 Esc：立即关闭弹窗。
  • 点击遮罩空白区：关闭弹窗。

这是前端常见高级交互，后续你可以把它复用到“编辑记录”“确认删除”等场景。
